Compiling from source on Windows

MinGW

Download and run the MinGW installer from http://www.mingw.org/download/installer. Use the default settings. When the package selector window comes up, you can quit the installer. The required packages will be fetched automatically...

Install JDK

download and run the JDK installer from http://www.oracle.com/technetwork/java/javase/downloads/index.html pick "jdk-8u45-windows-i586.exe" For developers, I recommend downloading and installing the Netbeans with JDK package from http://www.oracle.com/technetwork/java/javase/downloads/jdk-netbeans-jsp-142931.html

Get all dependencies

Double-click the "get_dependencies.bat" script in the platform_win directory. This downloads, extract, and compiles all necessary tools.
